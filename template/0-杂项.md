## 随机数

- Mersenne Twister

```c++
//随机种子
unsigned seed = std::chrono::system_clock::now().time_since_epoch().count();

//无范围
mt19937 rd(seed);
cout<<rd()<<endl

//2^64
mt19937_64 rd(seed);
cout<<rd()<<endl;

//手动设置范围
mt19937 rand_num(seed);  // 大随机数
uniform_int_distribution<long long> dist(0, 1000000000);  // 给定范围
cout << dist(rand_num) << endl;
```
